Components

Here are all components listed. Beneath the single compnonets are a "HTML"- and a "CSS"-Block, where the used Html-tags and defined classes can be found.

Ordred Lists

Ordred lists can be used to structure several elements from the same group hirachically:

  1. item
  2. item
  3. item
  4. item
                        
HTML:

                   
<ol class="list list-striped">
    <li>item</li>
    <li>item</li>
    <li>item</li>
    <li>item</li>
</ol>


                    
CSS:

                   
.list {
    & > li {
        padding: 10px;
    }
}

.list-striped {
    & > li:nth-child(odd) {
        background-color: #e5e5e5;
    }
}

Unordred Lists

Unordred list can be used to list up different elements from the same group:

  • item
  • item
  • item
  • item
                        
HTML:

                   
<ul class="list list-striped">
    <li>item</li>
    <li>item</li>
    <li>item</li>
    <li>item</li>
</ul>


                    
CSS:

                   
.list {
    & > li {
        padding: 10px;
    }
}

.list-striped {
    & > li:nth-child(odd) {
        background-color: #e5e5e5;
    }
}

Links

A common link provids to switch to another domain which is attached on a key word:

Link 
                        
HTML:

                   
<a class="link" href="https://..." > Link </a>


                    
CSS:

                   
.link {
    color: blue;
    text-decoration: none;
}

Clicking an action link means that something happens if it is clicked, like an "apply here"-link:

Link 
                        
HTML:

                   
<a class="link link-action" href="https://..." > Link </a>


                    
CSS:

                   
.link {
    color: blue;
    text-decoration: none;
}

.link-action {
    padding: 5px 40px 5px 40px;
    text-align: center;
    background-color: #e5e5e5;
}

Buttons

This Buttons can be rendered in different sizes and colors by using classes (see them in the css block below) 

                        
HTML:

                   
<button 
class="btn btn-big btn-color-blue"> Big Button </button>

<button 
class="btn btn-middle btn-color-blue">Middle Button</button>

<button 
class="btn btn-small btn-color-blue">Small Button</button>


                    
CSS:

                   
.btn {
    color: #EEEFF7;
    border: none;
    margin-bottom: 20px;
    display: flex;
}

.btn:hover {
    background-color: $highlightColor;
}

.btn-big {
    padding: 10px 50px 10px 50px;
}

.btn-middle {
    padding: 7px 30px 7px 30px;
    font-size: 14px;
}

.btn-small {
    padding: 4px 20px 4px 20px;
    font-size: 12px;
}

.btn-color-blue {
    background-color: $primaryColor;
}

.btn-color-light {
    background-color: $accentColor;
}

Forms

Good structured Formulars are very important, see below how it can be created:

Formular Write a header Write some Text 
                        
HTML:

                   
<form class="form">

    <fieldset>

        <legend 
            class="form-legend">
                Formular
        </legend>


        Write a header
        <input 
            type="text" 
            name="text" 
            class = "form-input-header" />


        Write some Text
        <textarea 
            rows="4" t
            ype="text" name="text" 
            class = "form-input-textarea" /> 
        </textarea>

        <label 
            for="option" 
            class="form-label">

            <input 
                id="option" 
                type="checkbox" 
                name="field" 
                value="option">
                    Value

        </label>

        <input 
            type="submit" 
            name="confirm" 
            value="OK" 
            class="btn btn-small">

    </fieldset>

</form>



                    
CSS:

                   
.form {
    padding: 15px;
    border-radius: 5px; 
    background-color: #e5e5e5;
    border: 1px solid transparent;
    box-shadow: 5px 5px 5px #888888;
}

.form-input-header {
    width: 100%;
    margin: 3px;
}

.form-input-textarea { 
    width: 100%;
}

Navigation

This is the Navigation and can be used with this classes:

  • First Page
  • Second Page
  • Last Page

It can be styled however you like:

  • First Page
  • Second Page
  • Last Page

Switching between several alignments is no problem anymore:

  • First Page
  • Second Page
  • Last Page 
                        
HTML:

                   
<nav class="navigation" id="navigation">
    <ul class="list-inline navigation-list list-centered">
        <li>
            <a href="#brand">First Page</a>
        </li>
        <li>
            <a href="#navigation">Second Page</a>
        </li>
        <li>
            <a href="#components">Last Page</a>
        </li>
    </ul>
</nav>



                    
CSS:

                   
.navigation {
    width: 100%;
    height: 60px;
    background-color: $darkColor;
    color: $lightColor;

    &.is-fixed {
        position: fixed;
        top: 0;
        left: 0;
    }
}

.navigation-list {
    & > li {
        a {
            display: inline-block;
            padding: 20px 15px;
            color: $lightColor;
            text-decoration: none;
        }

        &:hover {
            border-bottom: 2px solid $highlightColor;
        }
    }
}

.list-centered {
    display: block;
    text-align: center;
}

.list-inline {
    @extend .list-unstyled;

    & > li {
        display: inline-block;
        vertical-align: top;
    }
}
