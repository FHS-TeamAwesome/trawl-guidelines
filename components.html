Ordred Lists

Applying the 'list' class to a list element will change it's default styling. Using the 'list-striped' sub-class you can add a striped effect to your list.

  1. item
  2. item
  3. item
  4. item
HTML:
                   
<ol class="list list-striped">
    <li>item</li>
    <li>item</li>
    <li>item</li>
    <li>item</li>
</ol>

Unordred Lists

The same goes for unordered lists:

  • item
  • item
  • item
HTML:

    <li>item</li>

</ul>

Links

A common link can also be enhanced using the 'link' class:

Link
HTML:
                   
<a class="link" href="https://...">Link</a>

Clicking an action link means that something happens if it is clicked, like an "apply here"-link:

Link
HTML:
                   
<a class="link-action" href="https://..." >Link</a>

Buttons

This Buttons can be rendered in different sizes and colors by using different classes.

HTML:
                   
<button class="btn btn-big btn-color-blue">Big Button</button>

<button class="btn btn-middle btn-color-blue">Middle Button</button>

<button class="btn btn-small btn-color-blue">Small Button</button>

Media object

Use the media object as an abstract element to create flexible and more complex components like comments or blog posts.
You can set the 'media-left' or 'media-right' class to align the media element (e.g. image) inside the container.
